# Amazon Stock Time-Series Analysis

A statistical modeling project analyzing **Amazon (AMZN) stock performance** using financial indicators and regression techniques in **R**.

## Project Overview

This project evaluates how financial indicators relate to Amazon’s stock price over time.

Variables analyzed:

- Earnings Per Share (EPS)
- Book Value Per Share (BVPS)
- Observation Trend (OBS)

Methods included:

- Time-Series Analysis
- Correlation Analysis
- Linear Regression (OLS)
- Robust Regression
- Residual Diagnostics

## Key Findings

📌 EPS and BVPS showed a **strong positive correlation** with stock price.

📌 The OLS regression model achieved an **R² of 95.74%**, indicating excellent predictive fit.

📌 Robust regression identified **BVPS as the strongest consistent predictor** after accounting for outliers.
