# Amazon Stock Time-Series Analysis
Time-series and regression analysis of Amazon (AMZN) stock performance using financial indicators and robust modeling techniques in R.

## Methods Used

- Time-series visualization
- Histogram analysis
- Scatterplots
- Correlation analysis
- Linear regression
- Robust regression validation
- Residual diagnostics
